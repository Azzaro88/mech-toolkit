1 MAWP — Max Allowable Working Pressure 🛢️

  • Maximum pressure the vessel can safely withstand at design temperature.
  • The weakest component decides MAWP.
  • Usually higher than operating pressure.
Difference
Design Pressure → design input.
MAWP → actual allowable pressure after fabrication.
Code Ref: UG-98

2 DESIGN PRESSURE 📈

  • Pressure used for vessel design, including static head.
  • Normally selected slightly higher than operating pressure.
Example
Operating Pressure = 8 barg
Design Pressure = 10 barg
Code Ref: UG-21

3 DESIGN TEMPERATURE 🌡️

  • Metal temperature considered for design calculations.
  • Fluid temperature and metal temperature are not always the same.
Interview point
As temperature increases, allowable stress decreases.
Code Ref: UG-20

4 JOINT EFFICIENCY (E) 🔩

  • Ratio of welded-joint strength to solid-plate strength.
  • Full RT → E = 1
  • Spot / No RT → lower E
Used in — shell thickness
t = P·RS·E − 0.6P
Code Ref: UW-12

5 CORROSION ALLOWANCE (CA) 📏

  • Extra thickness to compensate for future corrosion or erosion.
  • Added over the required pressure thickness.
Example
Req. thickness = 12 mm · CA = 3 mm
Final thickness = 15 mm
Code Ref: UG-25

6 PWHT — Post Weld Heat Treatment 🔥

  • Heat treatment after welding to reduce residual stress.
Purpose
✓ Crack prevention   ✓ Hardness reduction   ✓ Hydrogen removal
Depends on
Thickness · Material · Service · Code requirement
Code Ref: UW-40 / UCS-56

7 MDMT — Min Design Metal Temp ❄️

  • Lowest safe operating metal temperature without brittle fracture.
  • Impact-test requirement is directly related to MDMT.
Code Ref: UCS-66

8 HYDROSTATIC TEST 💧

  • Pressure test using liquid to verify strength & leak tightness.
  • Hydrotest pressure ≈ 1.3 × MAWP.
Why water?
Water has low stored energy — safer than pneumatic testing.
Code Ref: UG-99

9 RADIOGRAPHIC EXAM (RT) 📡

  • NDT using X-rays or gamma rays for internal weld inspection.
  • RT level affects joint efficiency.
Detects
✓ Slag   ✓ Porosity   ✓ Lack of fusion
Code Ref: UW-51 / UW-52

10 REINFORCEMENT PAD (REPAD) ⭕

  • Extra plate around a nozzle opening for strength compensation.
Functions
✓ Area replacement   ✓ Stress distribution   ✓ Local reinforcement
Code Ref: UG-37 to UG-45
